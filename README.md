# IllustraterAutoPageCreater
テキストとJSONとテンプレートからイラストレーターで自動でページを作成する、イラストレーター用スクリプトです（漫トロ用）

##環境
###イラストレーター
`個人ランキングあてはめ.jsx``AIからPDFとPNGを一括で出力するやつ.jsx`をイラストレーターの外部スクリプトとして読み込んで実行します。

使用前にはそれぞれのスクリプトを参照して、適切なディレクトリ構造を作成してから使って下さい。

###ライブラリ
https://github.com/douglascrockford/JSON-js

のライブラリが必須です。json2.js等を同ディレクトリにコピーして使って下さい。
